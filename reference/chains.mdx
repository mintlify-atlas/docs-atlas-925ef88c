## Supported Networks

The x402r protocol is deployed on the following EVM chains:

### Mainnets

- **Ethereum** (Chain ID: `1`)
- **Base** (Chain ID: `8453`)
- **Polygon** (Chain ID: `137`)
- **Arbitrum One** (Chain ID: `42161`)
- **Optimism** (Chain ID: `10`)
- **Celo** (Chain ID: `42220`)
- **Avalanche C-Chain** (Chain ID: `43114`)
- **Monad** (Chain ID: `143`)
- **Linea** (Chain ID: `59144`)

### Testnets

- **Base Sepolia** (Chain ID: `84532`)
- **Ethereum Sepolia** (Chain ID: `11155111`)

## Network Identifiers

The SDK uses CAIP-2 format for network identifiers:

```ts
import { toNetworkId, fromNetworkId } from '@x402r/sdk'

const networkId = toNetworkId(8453) // "eip155:8453"
const chainId = fromNetworkId('eip155:8453') // 8453
```

## Checking Chain Support

```ts
import { isSupportedChain, getChainConfig } from '@x402r/sdk'

if (isSupportedChain(8453)) {
  const config = getChainConfig(8453)
  console.log(config.name) // "Base"
  console.log(config.usdc) // USDC token address on Base
}
```
