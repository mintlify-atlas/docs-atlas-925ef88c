<Warning>
  **Pre-release software** — The x402r SDK is not yet production-ready. Packages are currently at version `0.0.0` and have not been published to npm.
</Warning>

## Development Status

The SDK is under active development. Expect:

- **Breaking changes** without prior notice
- **API instability** across commits
- **Incomplete features** in certain modules

## Versioning

Once published, the SDK will follow [Semantic Versioning](https://semver.org/):

- **Major versions** (e.g., `1.0.0` → `2.0.0`) — breaking changes
- **Minor versions** (e.g., `1.0.0` → `1.1.0`) — new features, backward-compatible
- **Patch versions** (e.g., `1.0.0` → `1.0.1`) — bug fixes, backward-compatible
