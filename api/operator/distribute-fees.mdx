---
title: distributeFees
description: Distribute accumulated fees for a specific token
---

## Usage

```typescript
import { distributeFees } from 'x402r/actions'
import { walletClient } from './client'

const hash = await distributeFees(walletClient, {
  operatorAddress: '0x...',
  token: '0x...'
})
```

## Parameters

### walletClient
- **Type:** `WalletClient`
- **Description:** A viem wallet client with an active account for signing transactions

### parameters

<ParamField path="operatorAddress" type="Address" required>
  The address of the payment operator contract
</ParamField>

<ParamField path="token" type="Address" required>
  The token address for which to distribute accumulated fees
</ParamField>

## Returns

**Type:** `Promise<Hash>`

Returns the transaction hash of the fee distribution transaction.

## Example

```typescript
import { createWalletClient, http } from 'viem'
import { mainnet } from 'viem/chains'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const walletClient = createWalletClient({
  account,
  chain: mainnet,
  transport: http()
})

const txHash = await distributeFees(walletClient, {
  operatorAddress: '0x1234567890123456789012345678901234567890',
  token: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48' // USDC
})

console.log('Distribution transaction:', txHash)

// Wait for transaction confirmation
import { publicClient } from './client'
const receipt = await publicClient.waitForTransactionReceipt({ hash: txHash })
console.log('Fees distributed in block:', receipt.blockNumber)
```

## Notes

- Requires a wallet client with a connected account
- The transaction will distribute fees to the configured fee recipients
- Both operator fees and protocol fees are distributed in a single transaction
- Gas fees are paid by the account in the wallet client

## Error Handling

```typescript
try {
  const hash = await distributeFees(walletClient, {
    operatorAddress: '0x...',
    token: '0x...'
  })
} catch (error) {
  if (error.message.includes('account')) {
    console.error('No account connected to wallet client')
  } else {
    console.error('Fee distribution failed:', error)
  }
}
```
