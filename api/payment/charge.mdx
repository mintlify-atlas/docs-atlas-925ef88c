## Function Signature

```typescript
async function charge(
  walletClient: WalletClient,
  parameters: ChargeParameters
): Promise<Hash>
```

## Description

Captures previously authorized funds from the escrow contract and transfers them to the receiver. This action can only be executed after the escrow period has ended and requires the caller to have the appropriate permissions.

The `charge` action finalizes the payment, making the funds no longer refundable through the standard escrow mechanism.

## Parameters

<ParamField path="walletClient" type="WalletClient" required>
  Viem wallet client with an active account to sign and send the transaction
</ParamField>

<ParamField path="parameters" type="ChargeParameters" required>
  Configuration object for capturing funds

  <ParamField path="operatorAddress" type="Address" required>
    Address of the payment operator contract
  </ParamField>

  <ParamField path="paymentInfo" type="PaymentInfo" required>
    Payment information object matching the original authorization containing:
    - `operator`: Address of the payment operator
    - `payer`: Address of the account paying
    - `receiver`: Address receiving payment
    - `token`: ERC20 token address
    - `maxAmount`: Maximum authorized amount
    - `authorizationExpiry`, `refundExpiry`: Expiration timestamps
    - `minFeeBps`, `maxFeeBps`: Fee bounds in basis points
    - `feeReceiver`: Fee recipient address
    - `salt`: Unique payment identifier
  </ParamField>

  <ParamField path="amount" type="bigint" required>
    Amount of tokens to charge (must not exceed capturable amount)
  </ParamField>

  <ParamField path="tokenCollector" type="Address" required>
    Address of the token collector contract for the transfer
  </ParamField>

  <ParamField path="collectorData" type="Hex" required>
    Encoded data to pass to the token collector
  </ParamField>
</ParamField>

## Returns

<ResponseField name="hash" type="Hash">
  Transaction hash of the charge transaction
</ResponseField>

## Example

```typescript
import { charge } from '@x402r/core'
import { createWalletClient, http } from 'viem'
import { mainnet } from 'viem/chains'

const walletClient = createWalletClient({
  chain: mainnet,
  transport: http(),
})

const paymentInfo = {
  operator: '0x...',
  payer: '0x...',
  receiver: '0x...',
  token: '0x...', // USDC address
  maxAmount: 1000000n,
  preApprovalExpiry: 0,
  authorizationExpiry: 1735689600,
  refundExpiry: 1738368000,
  minFeeBps: 0,
  maxFeeBps: 500,
  feeReceiver: '0x...',
  salt: 0x123456789abcdefn,
}

// Wait for escrow period to end before charging
const txHash = await charge(walletClient, {
  operatorAddress: '0x...',
  paymentInfo,
  amount: 1000000n, // 1 USDC
  tokenCollector: '0x...',
  collectorData: '0x',
})

console.log('Charge transaction:', txHash)
```

## Requirements

- Escrow period must have ended
- Sufficient capturable amount must be available
- Caller must have permission to charge (typically the receiver or operator)
- Payment must not be frozen

## Error Handling

This function will throw an error if:
- Wallet client does not have an active account
- Escrow period has not ended
- Requested amount exceeds capturable balance
- Contract call fails or transaction reverts
